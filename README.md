# 🎬 剧本工坊

> 小说转剧本 · AI 智能转换工具 · 纯前端 · 无需部署

**剧本工坊** 是一款完全在浏览器本地运行的「小说 → 剧本」转换工具。粘贴小说原文，点击转换，即可自动生成结构化的剧本 YAML，支持场次划分、角色识别、对话提取、情绪标注。

---

## ✨ 功能一览

### 📝 智能转换
- **本地引擎** — 内置智能转换算法，无需任何配置，完全免费、离线可用
- **AI 模式** — 接入 OpenAI 兼容 API（支持 DeepSeek / 通义千问 / GPT 等），转换更精准
- **一键转换** — 粘贴小说 → 点击「开始转换」或按 `Ctrl+Enter` 快捷键
- **三种输出格式** — YAML（结构完整）、JSON（程序友好）、TXT（纯文本阅读）

### 🎭 角色管理
- 自动提取小说中所有角色
- 识别主角、主要角色、配角
- 统计台词句数，查看对话样本
- 支持角色改名和类型调整

### 📁 文件管理（文库）
- 导入 `.txt` 小说文件，自动分章
- 智能章节分割（按「第X章」标记或按字数）
- 章节列表折叠/展开，清爽不占空间
- 🔍 **章节搜索** — 快速按章节名定位内容
- 分章正则自定义

### 📑 原文 ↔ 剧本对照
- 左右对照视图，原文与剧本一一对应
- 场景颜色分块，视觉清晰
- 跨面板行级高亮（鼠标悬停自动匹配）
- 滚动同步
- 全屏沉浸浏览
- 右侧剧本支持**直接编辑**

### 📊 剧本展示（HTML PPT）
- 将剧本生成为 HTML 幻灯片
- 封面 → 角色介绍 → 每场一页 → 统计数据 → 结尾
- **9 种主题**（按 `T` 键切换）
- 键盘导航（← → 翻页 · `F` 全屏 · `O` 概览 · `S` 备注）

### 🎨 个性化
- **8 种页面主题**（极简白、柔和、暖黄、东京夜、摩卡、极光、日落等）
- 按 `T` 键快速循环切换主题
- 对照字体大小/字体自由调节
- 🌓 深色模式（全局反色）

### 💾 导入导出
- 导入 `.txt` 小说文件
- 导出 YAML / JSON / TXT
- 超过 5 幕自动拆分文件

## 🚀 快速开始

### 方式一：直接打开（推荐）

用浏览器打开 `index.html` 即可使用，**无需任何服务器**，所有数据存储在浏览器本地。

### 方式二：本地服务器（AI 模式需要）

```bash
# 使用 VS Code 的 Live Server
# 右键 index.html → Open with Live Server

# 或使用 Python
python -m http.server 8080

# 或使用 Node
npx serve .
```

> ⚠️ AI 模式需要从浏览器发请求到 API 端点，部分浏览器在 `file://` 协议下可能因 CORS 限制无法工作，建议用 Live Server 启动。

---

## 🧠 AI 模式配置

1. 点击「AI 模式」开关
2. 点击「API 设置」填写 API Key
3. 支持预设一键切换：
   - **DeepSeek** — `deepseek-chat`（性价比高）
   - **OpenAI** — `gpt-4o-mini`（效果稳定）
   - **通义千问** — `qwen-turbo`（国内首选）
4. 点击「测试连接」验证配置

## 📌 技术特点

- **纯前端** — 零依赖，不装包，不跑服务端
- **本地运行** — 所有处理在浏览器内完成，数据不上传（AI 模式除外）
- **localStorage 存储** — 小说文库、设置、API 配置持久化
- **智能分章** — 支持正则匹配章节标记 + 字数两种模式，自动检测编码（UTF-8 / GBK）
- **YAML 解析与生成** — 自实现轻量 YAML 解析器，无需第三方库
